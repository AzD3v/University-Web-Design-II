Introdução ao jQuery


O que é jQuery?

Essencialmente, jQuery é uma biblioteca da linguagem de programação JavaScript que tem por objetivo simplificar a produção de código da linguagem. Esse facto é bem patente no seu slogan: "Write less, do more".

Para escrever código em jQuery convém ter primeiro uma boa compreensão tanto da própria linguagem JavaScript em si (Vanilla JavaScript), bem como dos blocos que compõem a web, HTML & CSS.

Quanto às vantagens em utilizar esta biblioteca, há que referir:

  • A manipulação de HTML, CSS e da DOM é muito mais facilitada (como já seria de esperar);
  • jQuery obviamente também suporta eventos de HTML, bem como efeitos e animações;
  • Também possui capacicidade em utilizar AJAX (técnica utilizada param carregar informação no browser sem ser necessário recarregar a página);
  • Entre outros.

O suporte a jQuery poderá ser adicionado fazendo o download da biblioteca ou carregando-a remotamente através de uma CDN (Content Delivery Network), sendo esta uma técnica em que é possível aceder aos recursos pretendidos remotamente.

Sintaxe do jQuery


jQuery é uma biblioteca desenhada especificamente para selecionar elementos HTML e executar ações sobre esses precisos elementos.

Exemplo do uso de jQuery:

$(seletor).ação()

  • O símbolo do dólar, $, define o acesso à biblioteca;
  • O 'seletor' inquire os elementos HTML;
  • Por fim, a 'ação' é executada sobre o(s) elemento(s) HTML.

Exemplos concretos do uso de jQuery para manipulação de elementos HTML:

Exemplos Explicação
$("#elementoAEsconder").hide() Esconde o elemento com o id "elementoAEsconder"
$(".elementoAEsconder").hide() Esconde o elemento com a class "elementoAEsconder"
$("p").hide() Esconde todos os parágrafos do documento em questão
$(this).hide() Esconde o elemento ao qual se está a referir no momento
$('< p >< / p >') Cria um elemento de parágrafo

Seletores jQuery

O seletor jQuery ($) permite a pesquisa, seleção e manipulação de elementos HTML. É baseado nos atuais seletores de CSS, mas também possui seletores próprios.

Exemplos do uso do seletor de jQuery:

Uso do seletor jQuery Significado
$('< p >< / p >') Cria um elemento de parágrafo
$('p') Seleciona todos os elementos de parágrafo de um dado documento
$('#idASerSelecionado') Seleciona o elemento HTML que possui o id representado
$('.classeASerSelecionada') Seleciona todos os elementos HTML com a classe representada

Eventos jQuery


Como já foi referido, até mesmo na área em que apenas me referi ao Vanilla JavaScript, um evento é algo que acontece num determinado instante escolhido pelo programador. Como exemplos de eventos temos:

  • Mover o rato sobre um determinado elemento;
  • Selecionar um botão;
  • O clique num elemento;
  • Entre outros.
