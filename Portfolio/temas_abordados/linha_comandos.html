Interface da Linha de Comandos

Introdução à interface da linha de comandos


No início da cadeira de Web Design II, tanto eu como os meus colegas tivemos aulas tanto teóricas como práticas quanto ao manuseamento e própria interface da linha de comandos de um sistema operativo Linux, denominado de Ubuntu. Sendo óbvio que a linha de comandos também existe no mais popular sistema operativo Windows, a shell de um sistema operativo Linux é muito mais poderosa, flexível e versátil. De resto, individualmente falando, quando não estou a utilizar a partição de Ubuntu que possuo no meu computador pessoal, utilizo uma linha de comandos com diversas vantagens sobre a shell predefinida do Windows, sendo que o software que uso denomina-se Cmder.

Existem, de resto, dois tipos distintos de caminhos na árvore de diretórios de um sistema:

  • Caminhos absolutos: São os caminhos que partem a partir da raíz so sistema;
  • Caminhos relativos: São os caminhos que são construídos a partir de um determinado diretório de trabalho do sistema.


Irei apresentar nesta precisa secção uma tabela com os comandos principais passíveis de serem utilizados numa linha de comandos:


Descrição do comando Função do comando apresentado
ls Lista ficheiros e diretórios
ls -a Lista ficheiros e diretórios incluíndo os ocultos
mkdir ~nome~ Cria um diretório ~nome~
cd ~nome~ Muda para o diretório ~nome~
cd Muda para o diretório 'home'
cd .. Muda para o diretório anterior
pwd Apresenta o caminho de diretório atual
man ~comando~ Consulta o manual do comando especificado
cp 'ficheiro1' 'ficheiro2' Copia o 'ficheiro1' com o nome 'ficheiro2'
mv 'ficheiro1' 'ficheiro2' Move ou renomeia o 'ficheiro1' para 'ficheiro2'
rm 'ficheiro1' Remove o 'ficheiro1'
rmdir 'diretório' Remove um diretório vazio (necessita da opção "-r" para um diretório não-vazio)
cat 'ficheiro' Apresenta o conteúdo de um ficheiro
more/less 'ficheiro' Apresenta o conteúdo de um ficheiro paginado
head/tail 'ficheiro' Apresenta as primeiras/últimas linhas do ficheiro
grep 'regexp' 'ficheiro' Procura no conteúdo de um ficheiro por uma expressão

Considerações finais


  • A linha de comandos de um sistema operativo sempre me fascinou, mas nunca havia utilizado tanto uma e efetuado tamanha diversidade de operações na mesma;

  • Tanto nesta cadeira em particular como na cadeira de Sistemas Operativos, foi um prazer conhecer mais acerca de uma shell e descobrir constantemente que esta trata-se de uma ferramenta muito mais poderosa do que pensava anteriormente;

  • A verdade é que só executando comando após comando é que é possível perceber-se do quão versátil aquele suposto "background negro" que eu conhecia do sistema Windows pode ser.
